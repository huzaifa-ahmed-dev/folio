# Huzaifa Ahmed - Portfolio

This is a modern portfolio website built with [Next.js](https://nextjs.org), featuring 3D animations and interactive effects.

## Features

- **Next.js 14** App Router
- **React Three Fiber** for 3D visualizations
- **Custom Effects**: Antigravity particles, Glow trails, Floating orbs, Scan lines
- **Responsive Design**
- **Theme System** with dark mode support

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
